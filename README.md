# Практика по системной аналитике 

Цель работы – спроектировать сервис дл бронировани авиабилетов.

Проект выполнен в рамках производственной практики MEDIASOFT по системной аналитике.



### Репозиторий содержит:

- **openapi.yaml** – спецификация API сервиса в формате OpenAPI 
- **Практика MEDIASOFT.pdf** – документация для сервиса бронирования авиабилетов
- **/Диаграммы\** – папка с диаграммами докоментации
